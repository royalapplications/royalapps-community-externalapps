# `ExternalApps`

Setup and initialize external apps hosting.


## Type

```csharp
RoyalApps.Community.ExternalApps.WinForms.ExternalApps
```

## Methods

### `Cleanup(ILogger)`

Cleans up process-tracking services before the application host shuts down.


**Parameters**

- `logger`: An optional logger used to record cleanup details.

### `Initialize(ILogger)`

Initializes process-tracking services used by external app sessions.


**Parameters**

- `logger`: An optional logger used to record initialization details.
